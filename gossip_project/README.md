<h1> The Gossip Project </h1>


1. Installation : 


>
> bundle install
> 
> rails db:create
>
> rails db:migrate
> 
> rails db:seed
>
> rails s
>

Dans le navigateur aller à l'URL suivante : 

http://localhost:3000/


2. Connaître les versions utilisées : 

>
> rails about
>


3. Application

Composants utilisés

- FrameWork : Ruby on Rails

- base de donnée : PostGre 

- HTML/CSS : Bootstrap
